Uso Flask, un framework para desarrollar el servidor web en python.
Para instalar, ejecutar el script
$sh ./install.sh

Para correr:
$ sh ./run.sh

y luego en el navegador ir a localhost:5000 (en la pc donde ejecute el servidor)
Si desea abrir un browser en otra maquina, poner ipServidor:5000.
Por ejemplo 192.168.1.36:5000
Al hacer click en submit, la foto queda guardada en la carpeta upload
